Configurer l'application

Pour activer la synchronisation en temps réel entre tous vos postes, vous avez besoin d'une base de données Firebase gratuite. Suivez ces étapes une seule fois :

Étapes (10 minutes)

1
Allez sur console.firebase.google.com et connectez-vous avec un compte Google.
2
Cliquez Créer un projet, donnez-lui un nom (ex: suivi-equipe), désactivez Google Analytics, puis validez.
3
Dans le menu gauche, cliquez Realtime Database → Créer une base de données → choisissez Mode test → Activer.
4
Cliquez l'icône ⚙️ (Paramètres du projet) → Général → descendez jusqu'à Vos applications → cliquez </> pour ajouter une app web.
5
Copiez la valeur databaseURL qui ressemble à https://xxx-default-rtdb.firebaseio.com et collez-la ci-dessous.

Trouvée dans Paramètres du projet → Général → Configuration de l'application web

Firebase gratuit jusqu'à 1 Go de données et 10 000 connexions simultanées — largement suffisant.

Qui êtes-vous ?

Choisissez votre nom et une couleur. Vos collègues verront qui est connecté en temps réel.
